# 가상 의뢰서: 에어메이트 산업장비 카탈로그

## 의뢰 배경
저희는 공장 자동화와 설비 유지보수 고객을 대상으로 산업용 장비와 부품을 공급하는 가상의 B2B 유통사입니다.
현재 제품 정보가 엑셀 파일로만 관리되어 있어, 고객이 직접 제품을 검색하고 조건별로 찾을 수 있는 웹 카탈로그가 필요합니다.

## 목표
- 제품 40개를 검색·필터·정렬할 수 있는 공개형 카탈로그
- 제품별 고유 상세 페이지
- 모바일에서도 빠르게 사양을 비교해 볼 수 있는 화면
- 견적 문의로 연결되는 명확한 CTA
- 포트폴리오 시연용이므로 실제 결제·회원가입·관리자 기능은 제외

## 주요 사용자
- 제조업 구매 담당자
- 공장 설비·보전 담당자
- 자동화 설비 시공업체
- 산업용 장비 대리점

## 핵심 기능
1. 제품명·모델명·용도 통합 검색
2. 카테고리, 전압, 재고상태, 가격대 필터
3. 가격, 납기, 이름순 정렬
4. 제품 카드와 상세 페이지
5. 추천 제품 섹션
6. 견적 문의 폼 또는 문의 모달
7. 반응형 웹
8. 기본 SEO, sitemap, robots
9. CSV 기반 데이터 로딩
10. 가상 데이터 안내 문구

## 디자인 방향
- 산업용 B2B 느낌의 단정하고 신뢰감 있는 UI
- 네이비·슬레이트 기반, 포인트는 앰버 계열
- 과한 그라데이션과 AI 느낌의 장식은 피함
- 외부 이미지 없이 카테고리별 아이콘과 CSS 그래픽 사용
- 넓은 여백, 명확한 수치·사양 표현
